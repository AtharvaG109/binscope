# Changelog

All notable changes to this project will be documented in this file.

## [0.1.1] - 2026-04-19

Documentation and repository governance release.

### Added

- `CODEOWNERS` coverage for source, docs, tests, and GitHub config
- `docs/` landing page with example report artifacts
- SVG terminal snapshot for repository browsing
- committed sample terminal and JSON outputs generated from clean fixtures

## [0.1.0] - 2026-04-19

Initial public release.

### Added

- PE, ELF, and Mach-O binary analysis
- section entropy scoring and ASCII histograms
- suspicious import combination detection
- carved string extraction and categorization
- packer signature heuristics
- YARA skeleton generation
- JSON and terminal reports
- directory summarization mode
- fixture-backed integration tests
- GitHub CI and release automation
